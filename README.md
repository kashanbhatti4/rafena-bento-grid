# Rafena Bento Grid (רפאנא)

דף נחיתה יוקרתי, רספונסיבי ומעוצב המותאם במיוחד עבור מותג **רפאנא (Rafena)**.

דף נחיתה זה נבנה תוך דגש על חוויית משתמש (UX) יוצאת דופן, עיצוב פרימיום מודרני ותמיכה מלאה בכתיבה מימין לשמאל (RTL).

## 🚀 תכונות עיקריות (Key Features)

- **עיצוב Bento Grid יוקרתי**: פריסה מודרנית ואסתטית של רשת אלמנטים המציגה את מוצרי החברה בצורה נקייה ומקצועית.
- **סל קניות דינמי (Dynamic Cart Drawer)**: מערכת הוספה לסל המאפשרת הוספת מוצרים מתוך 12 מוצרי החברה השונים, כולל עדכון מחיר בזמן אמת, שינוי כמויות ותצוגה מונפשת.
- **סליידרים רספונסיביים ייעודיים לנייד**:
  - **קטגוריית רבי המכר (Best Sellers)**: סליידר מותאם לנייד עם מעבר טאבים דינמי המאפס את מיקום הסליידר.
  - **בלוג ומאמרים**: סליידר ייחודי לנייד התומך בגרירה/החלקה (Swipe).
  - **סיפורי הצלחה (Testimonials)**: סליידר המציג את חוות הדעת של הלקוחות בצורה חלקה.
- **חוויית ניווט רספונסיבית**: טאבים וקטגוריות בנייד המסודרים בשתי שורות (2x2) למניעת גלילה רוחבית מיותרת.
- **טיפוגרפיה ואנימציות מיקרו (Micro-animations)**: שימוש בפונטים מודרניים של Google Fonts, מעברים רכים (Transitions) ואייקונים אלגנטיים של Feather Icons.

## 🛠️ טכנולוגיות (Tech Stack)

- **HTML5**: מבנה סמנטי מותאם ל-SEO.
- **CSS3**: עיצוב CSS מתקדם, כולל Flexbox, Grid, משתני CSS (Custom Properties) ואנימציות חלקות.
- **JavaScript (Vanilla)**: לוגיקה נקייה, מהירה ועצמאית ללא תלות בספריות כבדות חיצוניות.

## 📂 מבנה הפרויקט (Project Structure)

- `index.html` - קובץ המבנה הראשי של דף הנחיתה.
- `index.css` - עיצוב הדף, הטיפוגרפיה והתאמות רספונסיביות מלאות למובייל ולדסקטופ.
- `index.js` - לוגיקת סל הקניות, הסליידרים, והחלפת הטאבים.
